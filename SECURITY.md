## Reporting a Vulnerability

We take the security of our software seriously. If you suspect you have found a vulnerability, please **do not** open a public issue.

### How to report

Please report vulnerabilities by emailing the maintainers or by using the ["Report a vulnerability"](https://github.com/Sipsaraa/question_gen_engine/security/advisories/new) feature on GitHub if enabled.

We will acknowledge your report within 48 hours, and proper steps will be taken to address the issue.
